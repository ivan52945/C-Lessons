# Переставить цифры
Переставить цифры в числе так, чтобы получилось максимальное число.

Данные на входе: 	`Одно целое не отрицательное число`  
Данные на выходе: 	`Целое неотрицательное число` 

## Пример №1
Данные на входе: 	`1229`  
Данные на выходе: 	`9221` 

## Пример №2
Данные на входе: 	`19`  
Данные на выходе: 	`91`
